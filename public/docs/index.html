Introduction

Rest Apis For Processing DemoTestInquires Using Job & Queues

Base URL: http://localhost:8000

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Endpoints

Store Demo Test Inquiry Payload

This endpoint is used to store a test inquiry payload to the system and then schedule a job to insert test payload items in the system.

Example request:
curl --request POST \
    "http://localhost:8000/api/demo/test" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"payload\": [
        {
            \"ref\": \"T-1\",
            \"name\": \"test\",
            \"description\": null
        },
        {
            \"ref\": \"T-2\",
            \"name\": \"test\",
            \"description\": \"Test description\"
        }
    ]
}"

const url = new URL(
    "http://localhost:8000/api/demo/test"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "payload": [
        {
            "ref": "T-1",
            "name": "test",
            "description": null
        },
        {
            "ref": "T-2",
            "name": "test",
            "description": "Test description"
        }
    ]
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Example response (201, success):


{
    "success": true,
    "data": null,
    "message": "Successfully Scheduled"
}

Example response (422, Some Validation Error || ref in db found with status inactive || Other Exception):


{
    "success": false,
    "data": null,
    "message": "some error msg"
}

Activate Test

This endpoint is used to activate a test.

Example request:
curl --request POST \
    "http://localhost:8000/api/demo/test/activate/T-2" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"ref\": \"laboriosam\"
}"

const url = new URL(
    "http://localhost:8000/api/demo/test/activate/T-2"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "ref": "laboriosam"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Example response (200, success):


{
